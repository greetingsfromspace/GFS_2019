Animation Test

This is a short test animation I did a few months ago using Toon Boom Harmony.

Long before Greetings From Space was even an idea, I used to make silly flash animations for fun (some of which I might share). This program is a lot more advanced, but easy enough to pick up if you've ever used Flash. I would recommend it to any would-be animators out there.

Unfortunately, I won't be creating an animated version of GFS any time soon. Far too time consuming for one man. That said, if FOX wants to dole out some cash, I'll put in the work... ;)
